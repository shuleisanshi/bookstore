### Project name:
Bookstore

### Introduction:
	这是一个练手的项目,主要是利用j2EE架构来实现网上购物的基本功能,虽然现在使用SSM架构和springboot架构比较多,但是j2EE还是基础,我们必须得牢固掌握,特别是和servlet相关. 

### Structure:
J2EE

### Skills:
MySQL, Servlet, Filter, Listener, javascript


### Function:
新用户注册及邮箱验证激活
登录及验证码
分页展示商品
购物车功能
查看订单详情
支付功能
